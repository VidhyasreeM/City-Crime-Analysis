🏙️ City Crime Analysis Dashboard
Project Overview

The City Crime Analysis Dashboard is a data analytics and visualization project developed using Power BI to analyze crime patterns, incident trends, and public safety insights within urban areas. The dashboard transforms raw crime data into interactive visual reports that help identify high-crime zones, crime frequency, and category-wise incident distribution.

This project demonstrates the use of Business Intelligence and analytical techniques to support data-driven decision-making in public safety and crime monitoring.

Objectives
Analyze city-wide crime trends and incident patterns
Identify high-crime areas and hotspots
Monitor crime frequency across different categories
Visualize time-based crime distribution
Generate insights to support public safety analysis

 Key Features
Interactive crime analysis dashboard
Crime category distribution visualization
Area-wise and location-based crime insights
Trend analysis using charts and graphs
KPI tracking for total incidents and crime rates
Dynamic filters and slicers for detailed analysis
Data-driven visual storytelling
